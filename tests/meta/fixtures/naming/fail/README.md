# Invalid Naming Examples

## Skills - WRONG
- skills/MySkill.md ✗ (should be in category/name/ structure)
- skills/analyze/parse.md ✗ (should be SKILL.md)
- skills/analyze/ParseRequirement/SKILL.md ✗ (PascalCase not allowed)

## Workflows - WRONG
- workflows/my-workflow.md ✗ (should be in developer/ or manager/)
- workflows/Developer/feature.md ✗ (Capital D not allowed)

## RFCs - WRONG
- docs/rfcs/my-proposal.md ✗ (missing number prefix)
- docs/rfcs/1-proposal.md ✗ (should be 001, 3 digits)
- docs/rfcs/001_proposal.md ✗ (underscore not allowed, use dash)
