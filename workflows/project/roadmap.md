# Project Roadmap Workflow

> Long-term planning and strategic alignment.

## Overview

| Aspect | Value |
|--------|-------|
| **Scope** | Organization/Project |
| **Trigger** | Annual planning or major pivot |
| **Output** | 12-month roadmap |
| **Duration** | Ongoing (reviewed quarterly) |

---

## Roadmap Phases

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   VISION    │ -> │   STRATEGY  │ -> │   ROADMAP   │
│  (Why)      │    │   (What)    │    │   (When)    │
└─────────────┘    └─────────────┘    └─────────────┘
```

## Phase 1: Vision Definition

### Context Mapping
| Skill Input | Source |
|-------------|--------|
| `context` | Market trends + company direction |
| `artifacts` | User research, competitive analysis |
| `goal` | Clear product/project vision |

**Call Skill:** `skills/analyze.md`

### Vision Components

1. **Problem Space**
   > {What problem are we solving? For whom?}

2. **Vision Statement**
   > {Where do we want to be in 1-3 years?}

3. **Success Metrics**
   | Metric | Current | 1-Year Target |
   |--------|---------|---------------|
   | {metric} | {value} | {target} |

## Roadmap Maintenance

### Quarterly Review

**Context Mapping:**
| Skill Input | Source |
|-------------|--------|
| `changes` | Completed initiatives |
| `context` | Market changes, strategy shifts |
| `standards` | Original vision & metrics |

**Call Skill:** `skills/review.md`

**Review Checklist:**
- [ ] Progress against roadmap
- [ ] Market changes impacting strategy
- [ ] Resource changes
- [ ] Priority adjustments needed

### Update Process

1. **Input Gathering**
   - Team feedback
   - Customer feedback
   - Market intelligence

2. **Prioritization**
   | Initiative | Impact | Effort | Priority |
   |------------|--------|--------|----------|
   | {init} | {H/M/L} | {H/M/L} | {score} |

3. **Stakeholder Alignment**
   - Present proposed changes
   - Gather feedback
   - Finalize roadmap

---

## Completion Criteria

- [ ] Vision clearly articulated
- [ ] Strategic themes defined
- [ ] 4-quarter roadmap created
- [ ] Dependencies mapped
- [ ] Stakeholders aligned
- [ ] Communication plan executed
