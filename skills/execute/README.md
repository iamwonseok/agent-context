# Execute Skills

**Purpose**: Perform the actual work.

## Skills

| Skill | Description | Role |
|-------|-------------|------|
| `write-code` | Write code using TDD | Developer |
| `refactor-code` | Refactor existing code | Developer |
| `fix-defect` | Fix bugs and defects | Developer |
| `update-documentation` | Update documentation | Developer |
| `manage-issues` | Manage issues in tracker | Both |

## When to Use

- Implementing features
- Fixing bugs
- Refactoring code
- Updating docs

## Typical Flow

```
Plan → write-code → Implementation
Bug Report → fix-defect → Fix
Tech Debt → refactor-code → Clean Code
```
