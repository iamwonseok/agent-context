# Integrate Skills

**Purpose**: Deliver results and integrate changes.

## Skills

| Skill | Description | Role |
|-------|-------------|------|
| `commit-changes` | Write commit message and commit | Developer |
| `create-merge-request` | Create merge request | Developer |
| `merge-changes` | Merge approved changes | Developer |
| `notify-stakeholders` | Notify stakeholders | Both |
| `publish-report` | Publish reports | Manager |

## When to Use

- After validation passes
- Submitting work for review
- Merging approved changes
- Communicating progress

## Typical Flow

```
Validated Code → commit-changes → Commit
Commits → create-merge-request → MR
Approved MR → merge-changes → Merged
Milestone → publish-report → Report
```
