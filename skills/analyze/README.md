# Analyze Skills

**Purpose**: Understand the situation before taking action.

## Skills

| Skill | Description | Role |
|-------|-------------|------|
| `parse-requirement` | Parse and clarify requirements | Developer |
| `inspect-codebase` | Analyze codebase structure | Developer |
| `inspect-logs` | Analyze logs for debugging | Developer |
| `evaluate-priority` | Evaluate task priority | Manager |
| `assess-status` | Assess project status | Manager |

## When to Use

- Starting a new task
- Debugging an issue
- Understanding existing code
- Evaluating priorities

## Typical Flow

```
User Request → parse-requirement → Clear Requirements
Error Report → inspect-logs → Root Cause
New Codebase → inspect-codebase → Understanding
```
